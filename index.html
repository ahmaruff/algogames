Algo Games


Algo Games adalah permainan berbasis web yang bertujuan untuk mengenalkan konsep algoritma pemrograman dengan menggunakan Visual Programming

Visual Programming memungkinkan kita membuat program, tanpa menggunakan metode konvensional (mengetik kode program). Melainkan menggunakan objek-objek visual seperti blok/gambar, yang dirangkai sedemikian rupa menjadi kode program yang valid.
